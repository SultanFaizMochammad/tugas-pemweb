# tugas-pemweb
## Description
This project is a web programming assignment (tugas pemweb) demonstrating basic web development concepts.

## Features
- Basic HTML structure
- CSS styling
- JavaScript functionality

## Installation
1. Clone the repository: `git clone https://github.com/SultanFaizMochammad/tugas-pemweb.git`
2. Open `index.html` in your web browser.

## Usage
Run the application by opening the HTML file in a browser. Modify the code as needed for your assignment.
